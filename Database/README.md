#### 조인

EMP table : 사원번호, 사원이름, 사원직책, 상관번호, 고용일, 월급, 커미션, 부서번호 등의 컬럼으로 구성됨
DEPT table : 부서번호, 부서이름, 부서위치 컬럼으로 구성됨

1. **Cross Join**

```java
SELECT *
FROM EMP, DEPT;
```
특정한 조건 없이 두 테이블 조인을 한 경우이다. 조건 없이 조인만 해서는 의미있는 데이터를 얻을 수 없다. 의미 있는 데이터를 얻기 위해 조건을 추가해서 조인을 해보자.
